# 📌 About The Project

Service Hub is a home services marketplace designed for Hyderabad residents. It allows users to discover trusted professionals, compare pricing, read reviews, and book services online.

The platform connects customers with service providers for electrical work, plumbing, cleaning, AC repair, painting, carpentry, appliance repair, and more.

# 🔄 Workflow

1. User registers or logs in.
2. User selects a service.
3. User chooses location.
4. User selects date and time.
5. Booking request is generated.
6. Service provider accepts booking.
7. Service is completed.
8. Customer submits review and rating.
